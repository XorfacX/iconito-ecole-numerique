Iconito est un outil libre géré par CAP-TIC. Vous pouvez trouver toute l'actualité d'Iconito sur nos deux sites web dédiés au projet:

Iconito.fr Le site officiel d'Iconito
Iconito.org Le site de la communauté des développeurs et des questions techniques.

Téléchargement

Adullact Les codes sources d'Iconito sont hébergés par l'Association des Développeurs et des Utilisateurs de Logiciels Libres pour l'Administration et les Collectivités Territoriales. Vous pouvez accéder à la page d'Iconito sur la Forge de l'Adullact

Sous le capot

Iconito utilise les technologies ou les produits suivants:

PHP et mySQL Les technologies de base d'Iconito
Copix Le framework de développement d'Iconito, intégrant notamment le modèle MVC, l'internationnalisation, les templates, les schémas XML de base de données, etc.
Prototype JS La librairie permettant de faire de l'Ajax et des transactions xml à la volée
SimpleViewer Visualisation en Flash des albums photos
DewPlayer Player MP3 des sons dans les blogs
DewSlider Diaporama d'images, en Flash
FCKeditor Editeur WYSIWYG
WikiRenderer Le premier formatage Wiki (de Laurent Jouanneau) d'Iconito
DokuWiki Un deuxième formatage Wiki, plus complet (voir syntaxe complète)
